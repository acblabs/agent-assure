# agent-assure v0.4.3

v0.4.3 adds control-coverage traceability maps and measured usage/cost-delta
evidence for evidence packets without turning framework mappings into scores,
grades, legal conclusions, or business impact claims.

## Added

- `agent-assure controls map PACKET.json --framework ... --out-dir ...`
  writes `control-coverage-report.json` and `control-coverage-report.md`.
- Built-in mappings cover `nist-ai-rmf`, `owasp-llm-top-10-2025`,
  `iso-iec-42001`, and `mitre-atlas-2026-06`.
- Reports include packet digest, mapping digest, conditional rule evaluations,
  coverage states, evidence references with optional `evidence_digest` values,
  and explicit limitations.
- Control-specific evaluated signals require packet-resident finding evidence;
  report generation does not infer passing controls from a summary rollup alone.
- MITRE ATLAS mappings pin release `2026.06` and include mapping strength plus
  tactic and technique IDs.
- Usage summaries can carry explicit pricing snapshot IDs, pricing snapshot
  content digests, cost-basis labels, and per-cost-observation denominators;
  comparison summaries can carry baseline usage, candidate usage, and
  deterministic integer basis-point usage deltas.
- `usage-pricing-snapshot` records versioned demo or caller-declared integer
  micro-USD token prices with optional cached-input and reasoning-token rates;
  the bundled demo pricing fixtures are marked as demo pricing, not live
  provider pricing.

## Schema

The active persisted schema version is now `0.4.3`, with a frozen
`schemas/v0.4.3` snapshot. Earlier schema directories remain available for
replay. v0.3.1 usage artifacts remain loadable, but v0.4.3-only usage fields
cannot be labeled as v0.3.1.

## Boundary

Control coverage reports map observed evidence for human review. Usage reports
record measured usage, declared estimated cost deltas only when declared pricing
provenance matches, and per-cost-observation evidence for review. They do not
judge framework conformance, organization-wide risk management, legal duties,
production threat resistance, or business impact.
