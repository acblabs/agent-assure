# agent-assure v0.4.2

v0.4.2 packages the sprint 7 through sprint 9 work: RAG provenance assurance,
counterfactual RAG robustness, and the experimental framework-adapter
foundation with a deep LangGraph integration.

Highlights:

- deterministic RAG provenance demo with digest-addressed policy chunks,
  cached vectors, corpus manifests, and `agent-assure demo rag`;
- counterfactual query-family fixtures that report query digests and required
  reference coverage without exposing raw query text or claiming semantic
  equivalence;
- experimental `agent_assure.adapters` foundation with a LangGraph translator,
  privacy-filtered framework observations, measured-usage attachment, and a
  bundled offline expense-assurance example;
- governance crosswalk visibility for NIST AI RMF, OWASP LLM Top 10,
  ISO/IEC 42001, and MITRE ATLAS, with explicit planning-scope boundaries;
- package metadata bumped to `0.4.2`.

This is a package release over the existing persisted artifact schema. Current
artifacts continue to emit `schema_version: 0.3.1`, and the active frozen
schema snapshot remains `schemas/v0.3.1`.
